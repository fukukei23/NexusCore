# ターミナルログのコピペを楽にする方法

## 最も簡単な方法

### `dev_tools/run_tests.sh` を使う
```bash
bash dev_tools/run_tests.sh tests/cli/test_bootstrap_apikey.py
```

**出力例**:

3. 開始マーカーから終了マーカーまでを選択してコピー

## その他の方法

### 出力をファイルに保存
```bash
pytest tests/cli/test_bootstrap_apikey.py -v | tee test.log
```
→ `test.log` を開いて必要な部分だけコピー

### 自動保存ログを参照
```bash
# 最新のエラーログを開く
cat $(ls -t docs/reports/TEST_ERRORS_*.txt | head -1)
```
