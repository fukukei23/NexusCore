# OpenCodeInterpreter WebUI

OpenAI API + Gradioによるマルチ機能対応のコード診断・修正・テスト・比較ツール。

## 主な機能

- コード診断・修正案・テスト自動生成
- 旧・新コードの意味的diff＋改善スコア
- ファイル/チャット履歴の一括DL（zip）
- 音声入力・Markdown/JSON対応

## セットアップ

1. `requirements.txt`で必要パッケージをインストール
2. `.env`にAPIキーを設定
3. `python src/opencodeinterpreter_webui.py`で起動
